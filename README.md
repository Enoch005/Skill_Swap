# SkillSwap

Learn What You Need. Teach What You Know.

## Project Description

SkillSwap is a skill exchange platform where users can teach skills they know and learn skills they need without spending money.

## Features

* User Registration
* User Login
* Profile Management
* Skill Management
* Skill Matching Engine
* Exchange Requests
* Ratings & Reviews
* Dashboard Analytics

## Technology Stack

### Frontend

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

### Backend

* Python
* Flask
